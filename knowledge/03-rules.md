# 搭+ 关键业务规则（提炼版）

> 完整规则详见飞书 PRD: `FkFZdljhJoJN1xx0QgfcqxgBnsf`

## 一、活动发布

- **状态流转**: 提交→已发布/未开始→可报名→已结束→已取消（V1.0人工审核→现已零审核）
- **发布前置条件**: 活动区域(省市区三级)强必填、活动开始≥当前时间、报名时间≤活动时间、敏感词检测、同主办方最多3个审核中活动
- **付费方式差异**:
  - 线上免费: 可修改所有字段
  - 线上付费: 非免费且有订单→仅可修改有限字段; 存在订单→无法修改价格
  - 线下活动: 标记线下标签，可修改所有字段
- **取消**: 主办方可取消→自动关闭报名→通知用户→自动退款
- **修改**: 活动开始前可修改(无订单时); 编辑权限仅限创建者

## 二、报名与支付

- **报名流程**: 已发布且报名未截止→选人数→填报名人信息→备注(100字)→免责声明(比赛)→微信支付→生成核销二维码→弹窗群二维码
- **限制**: 单笔订单人数上限按活动配置、报名人数实时扣减、超额自动限制
- **支付**: 前端计算总价→拉卡拉支付→成功跳转5秒后自动跳转订单→15分钟未支付自动取消→免费活动(金额0)仍生成二维码

## 三、核销

- **二维码**: 按报名人数生成等量加密二维码、每码状态独立(未核销/已核销/已退款/已过期)、含加密token唯一防伪、按价格规格独立生成
- **校验**: 合法性→所属活动→是否已核销→是否有效
- **推导**: 全部核销→已核销、部分核销+部分退款→部分完成、活动取消→全部已取消+二维码作废、活动结束未核销→过期未使用

## 四、售后与退款

- **申请条件**: 活动已结束、订单未结算、每单仅限一次
- **主办方处理**: 确认退款(全额/部分)→修改金额后确认→拒绝(需原因)→7天未处理→挂起→平台介入
- **执行**: 微信原路退回、1-3工作日到账、退款后二维码自动失效、支持按规格退款

## 五、财务结算（拉卡拉分账模式）

### 交易参与方
- **分账接收方（平台）**: 灵睿（搭+平台方），拉卡拉商户号 SR2024000131998
- **分账方（商户/主办方）**: 各入驻主理人，需完成拉卡拉商户进件
- **用户**: 微信支付付款方

### 分账流程
1. 用户通过微信支付将报名费付至拉卡拉中间账户，资金冻结
2. 活动结束后，平台发起分账指令：
   - 主办方 ~90%（扣除平台技术服务费后）
   - 平台 ~10%（技术服务费收入）
3. 分账完成后，资金进入各自拉卡拉账户
4. 主办方从拉卡拉账户提现至银行卡

### 费用与税务
- **交易手续费**: 2.5‰（万分之2.5），从平台分账部分扣除，由灵睿承担
- **平台收入确认**: 分账完成时点确认技术服务费收入（权责发生制）
- **增值税**: 平台按技术服务费收入缴纳 6% 增值税
- **发票**: 主办方向用户开具全额发票，平台向主办方开具技术服务费发票

### 退款流程
- 采用「先分账后冲正」模式
- 已分账订单退款：先执行分账回退（从主办方拉卡拉账户扣回），再原路退回用户

### 商户进件（已知痛点）
- 主理人入驻需完成 6+ 步拉卡拉进件流程：添加商户→实名认证→拉卡拉 APP 登录→微信扫码→申请分账权限→绑定分账接收账户
- 流程复杂、中小商户易流失，是当前业务扩展的主要瓶颈

## 六、身份与权限

| 操作 | 个人用户 | 认证主理人 | 平台管理员 |
|------|---------|-----------|-----------|
| 发布普通活动 | ✓ | ✓ | ✓ |
| 发布比赛 | ✗(弹窗引导认证) | ✓ | ✓ |
| 售后操作 | - | 仅PC端 | ✓ |
| 修改已发布活动 | 仅自己的 | 仅自己的 | 可强制操作 |

- 敏感信息: 身份证AES加密存储、后台脱敏(前4+后4)、API默认不返回id_card、日志禁止打印完整身份证

## 七、比赛业务

- **参赛组别**: 性别限定(不限/仅男/仅女)、年龄区间(系统根据身份证自动校验)、团队报名暂时不做
- **赛后**: 主办方上传成绩表(Excel)、C端查询成绩+浏览影集、电子完赛证书不需要
- **修改规则**: 存在订单的组别信息不可修改(需提示)

## 八、排行榜

- **周期**: T-30滚动窗口，每日凌晨00:00重算
- **公式**: 俱乐部榜=30天成功支付订单数/发布活动场次数; 用户榜=30天累计成功支付订单数; 前端展示分=真实热度+虚拟热度加成
- **干预**: 虚拟热度注入(后台手动填写默认0)、TOP3金银铜视觉强化、荣誉标签手动分配、黑名单不做

## 九、区域筛选

- **B端**: 省市区三级级联强必填、标准行政区划API禁止手动输入
- **C端**: 只有有在架活动的县区才显Tab、某区域<3个活动→缺省文案+跨区热门推荐填充、历史活动统一放全部下、仅对后续新活动生效

## 十、评价（待开发）

- **评分权限**: 仅已报名/已核销用户、三档满意度(不满意1/满意3/相当满意5)、快速标签+文字+图片
- **提醒**: 结束后120小时弹窗引导、同活动最多3次、连续2次稍后→30天免打扰、7天最多5次、超120小时永久保留入口
- **激励**: 50积分/次、连续3次解锁「体验达人」徽章
- **B端**: 主办方主页评价Tab、好评率/综合得分/标签云/评价列表、主办方可回复(每评价一次)
